## Reference: link-target priority

When the editor asks "what URL should we link to?", default in this order:

1. **/best-finviz-alternatives** — page-4 stuck, biggest ranking lift from any single backlink
2. **/scorecard** — proprietary data anchor, helps Dataset schema authority
3. **/** — brand-entity equity, helps the "tapeline" SERP problem
4. **/embed** — only if the article is specifically about free finance tools or developer resources
5. **/t/{relevant TICKER}** — if the article is about a specific stock the editor names

Avoid linking to `/signup`, `/app/*`, or `/pricing` — those are conversion pages, not link-equity pages.

## Reference: what to NEVER do

- Mass-blast identical emails (Google catches this, editors blacklist senders)
- Buy links (Google Penguin penalty)
- Comment-spam blog comment sections (zero DA value, looks desperate)
- Use link-exchange networks (PBNs) — Google has a dedicated penalty for this
- Forge affiliate relationships you don't have
- Ghost-write under fake personas — every link should be from a real Christian Piyatilaka outreach
